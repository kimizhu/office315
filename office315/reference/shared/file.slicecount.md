
# File.sliceCount property
Gets the number of slices into which the file is divided.

|||
|:-----|:-----|
|**Hosts:**|PowerPoint, Word|
|**Added in**|1.1|

```
var slices = file.sliceCount;
```


## Return value

The number of slices.
