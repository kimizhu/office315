
# Dictionary element
Defines settings for a task pane add-in that implements additional dictionary support.

 **Add-in type:** Task pane


## Syntax:


```XML
<Dictionary>
   ...
</Dictionary>
```

[OfficeApp](../../reference/manifest/officeapp.md)
